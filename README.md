## О проекте
IYHAN SHOP – это SPA интернет-магазин танков. Проект построен на Node.js + Express (бэкенд) и TypeScript (клиент). Проект предполагает авторизацию учётных записей для пользователей, обширный каталог танков, корзину выбранных позиций к покупке и её оплату с информацией о доставке.

## Добавление нового функционала
Для добавления своей части проекта необходимо выбрать свою ветку, после чего подхватить базовую структуру из ветки main.
1. Клонировать репозиторий: git clone https://github.com/<ваша учётка>/L_Shop.git
2. Перейти в ветку feature/auth и склонировать содержимое к себе в ветку
3. По ходу разработки добавляйте маршруты, функции и прочие элементы в существующие файлы/каталоги 

В базовую структуру каждой ветки входят:
- Корневые файлы:
	- package.json                    # зависимости проекта
	- tsconfig.json                   # настройки TypeScript для сервера
	- tsconfig.client.json            # настройки TypeScript для клиента
	- .gitignore                      # игнорируемые файлы
	- users.json                      # база пользователей (общая)
	- README.md 					  # документация

- Серверные файлы (server/):
	- controllers 					  # папка с логикой обработки запросов
	- middleware 					  # папка с промежуточными обработчиками
	- models 						  # папка с моделями данных
	- routes 						  # папка с маршрутами API
	- types 						  # папка с TypeScript типами
	- index.ts 						  # общие типы для сервера (User, Request и т.д.)

- Клиентские файлы (src):
	- pages 						  # папка со страницами приложения
	- services 						  # папка с сервисами
	- types 						  # папка с TypeScript типами
	- index.ts 						  # общие типы для клиента (User, AuthResponse)
	- main.ts 						  # точка входа клиента (запускает приложение)
	- router.ts 					  # главный роутер (управляет навигацией между страницами)
	
- Статические файлы (public):
	- index.html 					  # единственная HTML страница (корень SPA)
	- styles.css 					  # общие стили (кнопки, формы, карточки, навигация)

Каждый сотрудник репозитория при разработке должен:
- внедрить свои маршруты в server/index.ts
- внедрить маршруты своих страниц в src/router.ts
- внедрить свои страницы в src/pages
- внедрить свои методы в src/services/api.ts
- подключить дополнительные стили в public/index.html и css-стили отдельным файлов для каждой страницы
